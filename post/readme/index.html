校友會工作展望

Posted by 鄧啟行 on Wednesday, January 9, 2019

校友會工回顧與展望

不經不覺校友會已經歷廿多載，成立之時，校友們對民權校友會所舉辦之活動，初時心存懷疑，但經歷多年累積之口碑，慢慢由接受及後積極參與，這一切全賴各屆理事們盡心盡力服務校友們之成果，本人代表民權校友會在這深深感謝歷屆理事們之付出，以及熱心支持校友會之校友及老師們。

在最近幾年經歷社會事件及新冠疫情，以致本會所有活動被迫停辦，如今一切復常，希望本會在未來所舉辦之活動能得到各校友們繼續鼎力支持，藉以使校友間之情誼能更進一步。更希望藉此機會呼籲各有意服務校友們的校友，加入理事會的大家庭，共同延續本會為校友們聯繫感情的宗旨。
